# Análise de Jogadores - Exercício de Pandas/DataFrame

### Este projeto utiliza análise e manipulação de dados com a biblioteca Pandas no Python. O objetivo foi realizar uma Prova de Conceito (PoC) com um conjunto de dados de jogadores de futebol. 

### Consultas Realizadas

O script em Python executa as seguintes análises:

1. Salários Altos: Jogadores com salário superior a R$ 20.000,00.

2. Filtro Estadual: Nome e salário dos jogadores de times de Minas Gerais (MG).

3. Busca por Nome: Jogadores cujo nome contém a letra 'u'.

4. Ordenação por Salário: Lista de jogadores ordenada pelo salário (decrescente).

5. Ordenação Múltipla: Jogadores ordenados por Time (crescente) e Salário (decrescente).

6. Agregação por Time: Contagem total de jogadores por time.

7. Média Salarial: Cálculo da média salarial por time.
